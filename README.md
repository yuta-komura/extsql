# extsql

Java ソース内の `.append(...)` で組み立てた SQL 断片を抽出し、Oracle に流せる `INSERT` 文を `insert_extsql.sql` として出力する Node.js スクリプトです。

## 使い方

```cmd
node extsql.js <Javaソースのルートパス>
```

例:

```cmd
node extsql.js C:\\work\\project\\src
```

## 出力

`insert_extsql.sql`

## テーブル

```sql
CREATE TABLE SQL_APPEND_EXTRACT (
    ID            NUMBER GENERATED BY DEFAULT AS IDENTITY,
    FILE_PATH     VARCHAR2(1000),
    BLOCK_NO      NUMBER,
    SQL_TEXT      CLOB
);
```

## 免責事項

本ツールは Java ソース中の .append(...) 呼び出しから SQL らしき文字列断片を静的に抽出する簡易ツールです。  
実際にアプリケーションで実行される SQL を 完全に再現するものではありません。
